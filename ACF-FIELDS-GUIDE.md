# Редактирование контента через ACF

## Общие принципы

- Все редактирование — через админ-панель WordPress
- Поля ACF появляются при редактировании страниц/постов
- Изображения загружаются через Media Library
- Видео можно загружать через Media Library или вставлять ссылки
- Все группы полей хранятся в базе данных (кроме страницы настроек футера)

---

## Главная страница

**Страницы → Все страницы → Home**
Группа полей: **Главная страница** (8 полей)

| Поле | Тип | Описание |
|---|---|---|
| Hero Background Video | File (mp4, webm) | Фоновое видео в hero-секции |
| Hero Modal Video | File (mp4, webm) | Видео при клике на оверлей |
| Артисты в слайдере | Repeater | Слайдер артистов (изображение + артист + имя) |
| Talk-show изображения | Gallery (макс. 3) | Изображения для секции Talk-show |
| События | Repeater (макс. 3) | Постеры событий с кнопкой покупки |
| Мерч изображения | Gallery | Изображения для секции мерч |
| Ссылка Telegram (hero) | URL | Иконка Telegram под заголовком hero |
| Ссылка Instagram (hero) | URL | Иконка Instagram под заголовком hero |

### Артисты в слайдере (подполя)

- **Артист** — выбор из списка артистов (Post Object)
- **Изображение для слайдера** — отдельное изображение (не совпадает со страницей артиста)
- **Имя артиста** — текст, если нужно переопределить заголовок поста

### События (подполя)

- **Постер события** — изображение
- **Sold out** — галочка (если отмечена, вместо кнопки покупки — Sold out)
- **Текст кнопки** — например, «Купить билет»
- **Ссылка на билеты** — URL

## Talk-show

**Страницы → Все страницы → Talk-show**
Группа полей: **Talk-show** (4 поля)

| Поле | Тип | Описание |
|---|---|---|
| Видео 1 | File (mp4, webm) | Первое видео |
| Видео 2 | File (mp4, webm) | Второе видео |
| Ссылка «Больше выпусков» | URL | Кнопка внизу эпизодов (по умолчанию YouTube) |
| Эпизоды (до 4 карточек) | Repeater (макс. 4) | Карточки выпусков |

### Эпизоды (подполя)

- **Изображение** — обложка эпизода
- **Заголовок** — название выпуска
- **Подзаголовок** — краткое описание
- **Ссылка YouTube** — URL выпуска

---

## Артисты

**Артисты → Добавить новый / Все артисты**
Группа полей: **Страница артиста** (5 полей)

- **Изображение артиста** — главное изображение
- **Краткое описание** — отображается в списке артистов
- **Полное описание** — на странице артиста
- **Социальные сети** — Repeater (платформа + URL)
- **Треки** — Repeater (обложка + название + MP3)

Удаление артиста безвозвратное. Если артист был в слайдере на главной — удалите его оттуда вручную.

## DAWGS

**Страницы → Все страницы → DAWGS**
Группа полей: **DAWGS** (3 поля)

- **Главное изображение** — Image
- **Игроки** — Repeater (имя, описание, изображение)
- **Партнеры** — Repeater (название, изображение, описание)

---

## События

**События → Добавить новое / Все события**
Группа полей: **Событие** (6 полей)

- **Постер события** — Image
- **Sold out** — True/False
- **Текст кнопки** — Text
- **Ссылка на билеты** — URL
- **Дата события** — Date (опционально)
- **Локация** — Text (опционально)

## Футер

**Боковое меню → Футер**
Группа полей: **Футер — email и соцсети** (4 поля)

- **Email в футере** — отображается в футере и используется для mailto:
- **Соцсети в футере** — Repeater (название + ссылка)
- **Ссылка «Политика конфиденциальности»** — URL
- **Ссылка «Все права защищены»** — URL
